# Survivors Roguelike (7-Day Game Jam)

Built under a strict 7-day time limit, this project was driven by three main objectives: learning the Unreal Engine, collaborating effectively as a development team, and successfully delivering a working version of the game we envisioned.

## About the Game
This is a fast-paced, Vampire Survivors-style roguelike built entirely in Unreal Engine 5. Players must survive relentless hordes of enemies while leveling up and acquiring new abilities.
